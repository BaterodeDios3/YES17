Restrict User To Default POS

Now restrict the user to his default POS in Odoo!!

Odoo Restrict User To Default POS restricts the user to access his default POS only, that means other Point of Sale will not be visible to them. The access/visibility of other POS to the other users will be monitored by the admin. You can easily modify the visibility access as per your wish.

The additional security to the POS in Odoo!!
It happens when the POS can come into the hands of non POS users and misuse it. The high security of your POS store is a necessity to keep non users away from it and avoid frauds.
Odoo Restrict User to Default POS gives you the power to restrict the POS access for non-users. The access or the visibility of the POS can be controlled by you from the backend. This will help you to keep your POS store safe and avoid frauds and mismanagement too.
